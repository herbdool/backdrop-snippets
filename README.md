# README
## Visual Studio Code Backdrop Snippets
 
This extension adds rich language support for the PHP & Backdrop API to VS Code, including:

- Backdrop Tags & Snippets
 - Module system (Backdrop hooks)

# Using
Type part of a snippet, press enter and/Or tab, and the snippet unfolds.

![alt text](http://g.recordit.co/7Yi7bJ6t7T.gif "Snippets Preview")

# Installation

1. Install Visual Studio Code
2. Launch Code
3. From the command palette `Ctrl-Shift-P` (Windows, Linux) or `Cmd-Shift-P` (OSX)
4. Select Install Extension from Location
5. Type `ext install Backdrop-Snippets`
6. Choose the extension
7. Reload Visual Studio Code
